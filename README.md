filks
=====

Generates silly poems based off of mad-libbed, serious poems,
and a tagged text corpus.


Requirements
============

1. Python 3.3+

Installation
============

To install:

```
make install
```

Development
===========

To build the dev environment:

```
make venv
. venv/bin/activate
python main.py
```
